# Project name: LineBot_ManagerGroup

## How to work?
  
 - Prepare:
    - ngrok
    - vscode
    - nvm
    - node version: 12.22
	
 - Steps:
    1. Run ngrok
        - Get expose url.
    2. start your node js server
        - node index.js
    3. Verify webhook url on Line bot developer page.
  
 - `This is a message bot.`
